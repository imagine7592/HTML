19.웹서버운영

  1. 인터넷에서 bitnami 다운 받기

  2. bitnami manager 로 서버를 켜고 끌 수 있다

  3. 자신이 만든 html파일을 bitnami wampstack apache htdocs 안에 넣기

    • 2가지로 웹을 열 수 있는데 첫번째는 http://127.0.0.1/파일명.html 로 열기,
      두번째는 ctrl+o 파일 열기로 열 수 있다
      127.0.0.1/으로 접속하면 내 컴퓨터안의 파일들을 볼 수 있다

    • 127.0.0.1으로 접속하는 것은 웹 브라우저와 웹서버가 같은 컴퓨터에 있을 때 쓰는 것이고

    • 만약 웹 서버와 웹 브라우저가 다른 컴퓨터에 있을때 같은 네트워크를 공유한다면 네트워크 및 인터넷 설정에서 네트워크 및 공유센터에서 연결을 클릭한 뒤 자세히를 클릭하면
      ipV4의 주소를 알 수 있는데 이것으로 서버에 저장된 웹을 볼 수 있다


      • 웹호스팅 git hub는 말그대로 호스트역할로 서버를 대신 운영해주는것이고,
      bitnami는 내가 직접 웹서버가 되어 웹브라우저를 제공해주는것
